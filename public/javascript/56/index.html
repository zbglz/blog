正则表达式简介

正则表达式用于定义一些字符串的规则。计算机可以根据正则表达式，来检查一个字符串是否符合规则。或者将字符串中符合规则的内容提取出来。

语法： var 表达式变量名 = new RegExp(‘正则表达式’,’匹配模式’)
检查： 表达式变量名.test(字符串)
结果： 返回true说明字符串符合该正则表达式，返回false说明不符合

在正则表达式的构造函数中，可以传入一个匹配模式作为第二个参数。

i:忽略大小写
g:全局匹配模式

这是正则表达式最简单的匹配规则，找什么就写什么。
