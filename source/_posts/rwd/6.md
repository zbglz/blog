<!DOCTYPE html>
<html>
  <head>

    <!-- 图标 -->
    <link rel="stylesheet" type="text/css" href="link/fontAwesome/css/all.min.css" />
    <title>响应式菜单iamge</title>
    <style>
      * {
        padding: 0;
        margin: 0;
      }

      body {
        text-align: center;
      }

      h3,
      p {
        padding: 20px;
      }

      #container {
        width: 960px;
        margin: 0 auto;
      }

      #header,
      #nav,
      #footer,
      #mainbody,
      #sidebar {
        line-height: 25px;
        background: #eee;
      }

      #mainbody,
      #sidebar {
        height: 300px;
        float: left;
      }

      #mainbody {
        width: 620px;
        margin-right: 20px;
        background: #ddefcb;
      }

      #sidebar {
        width: 320px;
        background: #ffcbdd;
      }

      #footer {
        clear: both;
      }

      /* 导航菜单 */
      #nav {
        background: none;
      }

      ul {
        list-style: none;
        padding: 0;
      }

      #nav li,
      #nav li a {
        display: inline-block;
        padding: 7px 15px;
        text-decoration: none;
        color: #3b3b3b;
      }

      #nav li a:hover {
        background: #eee;
      }

      #nav .active {
        background: #eee;
      }

      .toggle-btn {
        display: none;
      }

      /*img 标签设置100%*/
      #showcase img {
        width: 100%;
        max-width: 100%;
        /*保证图片不被拉伸*/
        vertical-align: middle;
      }

      #backgroundImg {
        background: url(link/test.jpg) 50% 50% no-repeat;
        -webkit-background-size: 100%;
        background-size: 100%;
        /* 背景图片的高宽比大约为0.4815 */
        padding-top: 48.15%;
      }

      /****** 在样式表创建一个可视窗口大于1200像素的媒体查询 */
      @media (min-width:1200px) {
        #container {
          width: 1170px;
        }

        #mainbody {
          width: 770px;
          margin-right: 30px;
        }

        #sidebar {
          width: 370px;
        }
      }

      /****** 当页面宽度小于等于959像素时，(以960为参照,页面的设计就是960px) */
      @media (max-width:959px) {
        #container {
          width: 100%;
        }

        #mainbody {
          width: 67%;
          /*教程中没有，纠错：继续缩小窗口会使内容区和边栏区错位*/
          margin-right: 0;
        }

        #sidebar {
          width: 30%;
          /* 剩下的30%可以作为两个栏目之间的空隙，直接将右侧栏浮动到右侧 */
          float: right;
        }
      }

      /*****为小尺寸设备设置断点，ipad在垂直的状态下，宽度是768px，可以设置在这个媒体宽度以下的媒体查询，一般智能手机在垂直或者水平方向的宽度都会小于这个尺寸。*/
      @media (max-width:767px) {

        #mainbody,
        #sidebar {
          float: none;
          width: 100%;
        }

        /* 导航切换图标 */
        .toggle-btn {
          display: inline-block;
          font-size: 20px;
          text-decoration: none;
          background: #000;
          color: #FFF;
          padding: 7px 10px;
          position: absolute;
          right: 8px;
          top: 12px;
        }

        /*隐藏关闭图标*/
        #nav .close {
          display: none;
        }

        /*这里呢设置，当导航为目标状态的时候显示关闭按钮*/
        #nav:target .close {
          display: inline-block;
        }

        /*将导航菜单的初始状态设置为隐藏*/
        #nav {
          display: none;
        }

        /*当导航菜单为目标状态时设置为显示*/
        #nav:target {
          display: block;
          position: absolute;
          top: 0;
          left: 0;
          padding: 64px 0px 0 0px;
          width: 100%;
          box-sizing: border-box;
        }

        /*当导航菜单为目标状态时，ul及内部其他元素的样式*/
        #nav:target ul {
          background: #000;
        }

        #nav:target ul li,
        #nav:target ul li a {
          display: block;
          color: #eee;
          text-align: left;
        }

        #nav:target ul li a:hover {
          background: #3b3b3b;
        }

        #nav:target .active {
          background: #272727;
        }

      }
    </style>
  </head>
  <body>
    <div id="container">

            style="vertical-align: middle;"></i></a>

      <div id="showcase"><img src="link/test.jpg" alt="" /></div>
      <p>背景图片自适应，因为承载背景图元素的高度无法确定（元素的高度需要根据背景图片的缩放，高度变化跟着变化），所以我们需要计算背景图片的高宽比，然后设置padding-top即可%。</p>
      <p>当然不是所有的图片都有相同的高宽比，可以设置background-size:cover,使图像可以自动按比例填满背景。</p>
      <div id="backgroundImg"></div>
      <section id="mainbody">
        <h3>主体</h3>

        </p>
      </section>
      <aside id="sidebar">
        <h3>边栏</h3>
