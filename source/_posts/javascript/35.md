###  函数和方法的区别

> 函数

直接定义的函数就是函数。


    <script>
      function sum(){
        console.log("这是一个函数")
      }
    </script>


> 方法

对象中的函数就是方法。

我们知道对象的属性可以是任意类型，当然也可以是一个函数，这种作为对象属性的函数，被称之为这个对象的方法method。


    <script>
      var obj = new Object()
      obj.name = "zhangsan"
      obj.run = function(){
        console.log("这是一个方法")
      }
    </script>


调用对象中的这个函数，**称之为**调用这个**对象的方法**！


    <script>
      var obj = new Object()
      obj.name = "zhangsan"
      obj.age = 18
      obj.run = function(){
        console.log("zhangsan is running")
      }
      obj.run() // 调用对象方法
    </script>


**函数和方法这只是名称上的区别，没有本质上的差别**。


    比如之前用到的 console.log()就是调用console对象的log方法
    document.write()就是调用document对象的write方法
    除此之外，他们还有很多其他的方法。
