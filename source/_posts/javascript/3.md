### 数据类型

* 变量可以定义的数据类型、或者说字面量有哪些数据类型。
* 例如：变量名、函数名、属性名，都属于标识符。
* 在JS中一共有6种数据类型。


```js js
<script>
  var a = "hello"; // 字符串：String
  
  var b = 123;  // 数值：Number
  
  var c = true;  // 布尔值：Boolean
  
  var d = null;  // 空值：Null
  
  var e = undefined;  // 未定义：Undefined
  
  var f = {};  // 对象：Object
  
  console.log(a,b,c,d,e,f)
</script>
```


* 其中，前五种属于基本数据类型。而Object属于引用数据类型。
