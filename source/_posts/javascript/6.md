### 布尔值：Boolean

Boolean只有两个值，一个true，一个false。true表示逻辑上的真，false表示逻辑上的不成立，为假。


```js js
<script>
  var bool = true;
  console.log("bool的值是：" + bool); 
  console.log("bool的值类型是：" + typeof bool); 
  console.log("-----------------")
  
  bool = false;
  console.log("bool的值是：" + bool); 
  console.log("bool的值类型是：" + typeof bool); 
</script>
```
