### test

只有页面

```js es6
<!-- index.html -->
<link rel="stylesheet" href="page.css">
<body>
  <div>123</div>
</body>

<!-- page.css -->
<style>
body{
  background-color:lightblue;
}
</style>
```

只有js

```js es6
<!-- a1.js -->
<script>
console.log("jsonly")
</script>
```

有页面也有js、css

```js es6
<!-- index.html -->
<link rel="stylesheet" href="page.css">
<script type="text/javascript" src="a1.js"></script>
<body>
  <div>123</div>
</body>

<!-- a1.js -->
<script>
console.log("有页面也有js、css")
</script>

<!-- page.css -->
<style>
body{
  background-color:lightblue;
}
</style>
```
